## 🚀 Sobre mí

Soy ingeniera de sistemas, apasionada por la tecnología, enfocada en el desarrollo de software con impacto real para negocios y personas.
Me encanta resolver problemas reales con soluciones prácticas y accesibles, manejando en paralelo dos stacks completos: **Next.js / NestJS / Prisma / PostgreSQL** y **React / Django REST Framework**.

Actualmente trabajo en proyectos que combinan desarrollo full-stack, SaaS, inteligencia artificial y desarrollo móvil:

- 🏢 Co-fundadora de **B-Mon System**, empresa de software dedicada al desarrollo y venta de sistemas web y productos SaaS para negocios en Perú.
- 🐾 Creadora de **PawSoft**, SaaS multi-tenant para gestión de centros veterinarios (historias clínicas, citas, facturación electrónica SUNAT, recordatorios por WhatsApp).
- 🛒 Desarrolladora de **SysVenta**, sistema de gestión de ventas, compras e inventario con facturación electrónica y roles de usuario.
- 🦷 Co-desarrolladora de **NovaDent (DentalClinic Pro)**, sistema de gestión para clínicas dentales (odontograma FDI, anamnesis, agenda, contratos).
- 📄 Co-desarrolladora de **CVManager**, sistema de gestión de currículums vitae para instituciones educativas.
- 🧠 Autora de tesis en deep learning: detección de nódulos pulmonares malignos en tomografías mediante redes neuronales convolucionales 3D (AUC-ROC 0.9069).
- 🚑 Creadora de **GeoAmbulancias**, sistema que optimiza la gestión de emergencias con geolocalización.
- ✋ Desarrolladora de un juego educativo móvil que enseña lengua de señas (imágenes, videos, cámara).
- 🤖 Creadora de un asistente de IA local estilo JARVIS (Ollama + Python), con voz, búsqueda en internet y clima en tiempo real.

## 🌟 Proyectos destacados

📍 Todos mis proyectos los encuentras aquí en GitHub. Aquí te comparto algunos:

🔹 **B-Mon System**
🏢 Landing page y panel administrativo para mi empresa de software (dashboard, productos, clientes, pagos y contratos).
🛠️ Next.js, NestJS, Prisma, PostgreSQL, Tailwind CSS

🔹 **PawSoft**
🐾 SaaS multi-tenant para centros veterinarios: historia clínica, citas, facturación SUNAT (Nubefact), inventario y recordatorios automáticos.
🛠️ NestJS, Prisma, PostgreSQL, Next.js, Tailwind CSS, BullMQ, Twilio

🔹 **SysVenta**
🛒 Sistema de gestión de ventas, compras e inventario con facturación electrónica y roles (ADMIN, CAJERO, ALMACENERO).
🛠️ NestJS, Prisma, PostgreSQL, Next.js, React

🔹 **NovaDent — DentalClinic Pro**
🦷 Sistema de gestión odontológica: odontograma FDI, anamnesis, agenda, contratos con pagos.
🛠️ NestJS, Next.js, TypeScript

🔹 **CVManager**
📄 Sistema de gestión de currículums vitae para una institución educativa.
🛠️ Laravel, PHP, Blade

🔹 **Deep Learning para detección de cáncer pulmonar** (Tesis)
📄 Detección de nódulos pulmonares malignos con redes neuronales convolucionales 3D (VideoResNet R3D-18), transfer learning, dataset LIDC-IDRI. AUC-ROC 0.9069.
🛠️ Python, PyTorch, Gradio, Grad-CAM

🔹 **GeoAmbulancias**
🗺️ Sistema de gestión de ambulancias con geolocalización y acceso a datos oficiales.
🛠️ PHP, MySQL, API Google Maps, Integración RENIEC

🔹 **Juego educativo de lengua de señas**
🎮 Aplicación móvil que enseña lengua de señas usando imágenes, videos y cámara.
🛠️ Android Studio, Java, XML
